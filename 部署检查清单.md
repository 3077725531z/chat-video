# ✅ 部署前检查清单

在部署前，请确认以下项目：

## 📋 代码检查

- [ ] 所有文件都已准备好
- [ ] `package.json` 包含所有依赖
- [ ] `server.js` 可以正常运行
- [ ] 没有语法错误
- [ ] 本地测试通过

## 📦 文件检查

- [ ] `package.json` 存在
- [ ] `server.js` 存在
- [ ] `public/` 文件夹存在
- [ ] `public/index.html` 存在
- [ ] `public/app.js` 存在
- [ ] `public/style.css` 存在
- [ ] `.gitignore` 存在
- [ ] `render.yaml` 存在（可选）

## 🔧 配置检查

- [ ] `package.json` 中有 `start` 脚本
- [ ] `package.json` 中指定了 Node 版本
- [ ] 端口使用 `process.env.PORT`
- [ ] 健康检查端点 `/health` 已配置

## 🚀 部署准备

- [ ] GitHub 账号已注册
- [ ] GitHub 仓库已创建
- [ ] 代码已上传到 GitHub
- [ ] Render 账号已注册（或准备注册）

## ✅ 部署后检查

- [ ] 应用可以访问
- [ ] 聊天功能正常
- [ ] 视频通话功能正常
- [ ] 移动端显示正常
- [ ] HTTPS 正常工作

## 🎯 完成！

如果所有项目都打勾，就可以开始部署了！

参考 `README-部署.md` 进行部署。
