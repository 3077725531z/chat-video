==========================================
  提交代码到GitHub - 快速命令
==========================================

在项目文件夹中打开命令行，依次执行：

1. 添加所有修改：
   git add .

2. 提交代码：
   git commit -m "优化视频功能和界面"

3. 推送到GitHub：
   git push

==========================================
如果提示需要登录，使用Personal Access Token
==========================================

生成Token：
https://github.com/settings/tokens
