# 🚀 一键部署指南（最简单方法）

## 方法一：使用Render（推荐，5分钟部署）

### 第一步：上传代码到GitHub

1. **如果没有GitHub账号，先注册**：https://github.com

2. **创建新仓库**：
   - 登录GitHub
   - 点击右上角 "+" → "New repository"
   - 仓库名：`chat-video-app`
   - 选择 Public（公开）
   - 点击 "Create repository"

3. **上传代码**（在项目文件夹中执行）：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/chat-video-app.git
   git push -u origin main
   ```

   如果不会用Git，也可以：
   - 在GitHub网页上直接上传文件
   - 点击 "uploading an existing file"

### 第二步：部署到Render

1. **访问Render**：https://render.com
   - 点击 "Get Started for Free"
   - 使用GitHub账号登录（最简单）

2. **创建Web Service**：
   - 点击 "New +" → "Web Service"
   - 选择 "Connect GitHub"
   - 授权Render访问你的GitHub
   - 选择你的仓库 `chat-video-app`

3. **配置设置**（自动检测，通常不需要修改）：
   - Name: `chat-video-app`
   - Environment: `Node`
   - Build Command: `npm install`（自动填充）
   - Start Command: `npm start`（自动填充）
   - Plan: `Free`

4. **点击 "Create Web Service"**

5. **等待部署**（5-10分钟）
   - 可以看到实时日志
   - 部署成功后会有绿色的 "Live" 标志

6. **获取访问地址**：
   - 部署成功后，Render会给你一个URL
   - 例如：`https://chat-video-app.onrender.com`
   - 这个就是你的应用地址！

### 完成！🎉

现在你可以：
- 在任何地方访问你的应用
- 在手机上打开链接使用
- 分享给朋友一起聊天视频

## 方法二：使用Railway（更快速）

### 第一步：上传代码到GitHub
（同方法一）

### 第二步：部署到Railway

1. **访问Railway**：https://railway.app
   - 点击 "Start a New Project"
   - 使用GitHub账号登录

2. **部署**：
   - 选择 "Deploy from GitHub repo"
   - 选择你的仓库
   - Railway会自动检测并部署
   - 等待2-3分钟

3. **获取地址**：
   - 部署完成后会显示URL
   - 例如：`https://chat-video-app.up.railway.app`

### 完成！🎉

## ❓ 常见问题

### 1. 部署失败怎么办？
- 检查代码是否完整上传到GitHub
- 查看部署日志中的错误信息
- 确保 `package.json` 正确

### 2. 应用打不开？
- Render免费版在15分钟无活动后会休眠
- 首次访问需要等待30秒左右唤醒
- 这是正常现象

### 3. 视频无法连接？
- 确保使用HTTPS（所有部署平台都提供）
- 检查浏览器控制台的错误
- 确保摄像头/麦克风权限已允许

### 4. 消息没有保存？
- Render会保留文件，但重启后会重置
- 如果需要永久存储，可以考虑使用数据库
